XtreamNew
- تحسينات كبيرة فى السرعة وسلاسة التنقل داخل جميع أقسام البلجن.
- إضافة صفحة ويب لإدارة السيرفرات والتحكم بها ومعرفة ال ping وحالة السيرفر وغيرها http://ip:8090
- إضافة Maintenance Center داخل شاشة الضبط لإدارة البلجن بالكامل:
تنظيف الكاش.
حذف الملفات المؤقتة.
حذف صور الـ Artwork والكاش.
- إضافة اللغة الفرنسية إلى TMDB.
- إضافة اللغة الفرنسية إلى محرك الترجمة Subsource.
- اضافةخيارات جديدة فى الضبط داخل Tmdb لترجمة النوع والبلد والوقت عن طريق قاموس بدلا من ترجمة Tmdb تأتى اخيانا الكتابة غلط ومع عدة خيارات
- ترجمة النوع Genre Translation
- نرجمةالبلد Country Translation
- ترجمة الوقت Runtime
- إمكانية اختيار مكان حفظ ملف Settings من شاشة الضبط عن طريق زر Menu.
- دعم NewVirtualKeyboard أثناء البحث فى الترجمة مع إمكانية اختيار لوحة المفاتيح من شاشة الضبط.
- تحسين واجهة المستخدم وتوحيد الألوان فى جميع شاشات البلجن.
- إضافة Runtime داخل صفحات الأفلام والمسلسلات والحلقات.
- تحسينات عامة وإصلاح عدد كبير من الأخطاء.

- تسجيل مباشر من القنوات عن طريق زر INFO اثناء تشغيل القناة.

- ثانى خيار XtreamEPGGrabber assigned XML لو عايز تظهر القنوات اللى عامل ليها assign فقط من بلجن EPGGrabber
- ثالث خيار Hybrid: Assign first + Server fallback لو عايز تظهر epg السيرفر + Epg assign كدة هتضمن كل القنوات يكون ليها Epg
- Replay / Catch-up عند دعم السيرفر عن طريق زر EPG
- تصدير روابط القنوات إلى IPAudio عن طريق زر 5.
- قائمة جانبية سريعة للتنقل بين الأقسام عن طريق زر 1
